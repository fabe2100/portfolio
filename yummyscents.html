Yummy Scents - E-commerce Webstore Development

This project involved the design and development of "Yummy Scents," a responsive e-commerce webstore specializing in gourmand perfumes with natural ingredients. The objective was to create an engaging and user-friendly online shopping experience, optimized for various devices.

Image: Project Planning (Replace 'image_project_planning.jpg' with your image file)

Key Skills Demonstrated

  • Responsive Web Design: Developed a fully responsive design using HTML, CSS, and media queries to ensure optimal display across desktops, tablets, and smartphones.
  • Front-End Development: Utilized HTML and CSS for structure and styling, incorporating modern techniques like Flexbox for layout management.
  • User Experience (UX) Design: Focused on creating intuitive navigation, clear calls to action, and engaging product presentation to enhance the user's shopping journey.
  • Visual Design: Implemented a cohesive visual identity, including color palette, typography (Playfair Display, Roboto), and imagery, to create an appealing and on-brand aesthetic.
  • E-commerce Functionality: Integrated essential e-commerce features such as product displays, shopping cart notifications, and detailed product descriptions.
  • Accessibility: Addressed web accessibility by implementing proper color contrast and addressing WAVE tool errors to ensure a better user experience.
  • Performance Optimization: Utilized Google PageSpeed Insights to analyze and optimize website performance, including image compression (webp) to improve loading times.
  • AI Tools Integration: Leveraged AI tools to generate product images, descriptions, and assist with JavaScript coding, demonstrating efficient use of technology in the design process.

Design Process

  1. Wireframing: Started with low-fidelity wireframes to plan the site's structure and navigation, followed by high-fidelity wireframes for detailed design visualization. [cite: 32, 33, 38, 39]
  2. Visual Identity: Developed a unique visual identity incorporating a custom logo, color palette, and typography to reflect the brand and appeal to the target audience. [cite: 34, 35, 36, 37]
  3. Development: Built the webstore using HTML, CSS, and JavaScript, with a focus on responsive design and user-centered functionality. [cite: 52, 53]
  4. Testing and Optimization: Conducted thorough testing for code validation, accessibility (WAVE tool), performance (Google PageSpeed Insights), and cross-browser compatibility. [cite: 28, 29, 30, 81, 85]
