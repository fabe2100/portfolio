Image Processing for Web

Throughout the project, attention was paid to digital image processing techniques to ensure images were suitable for web use, including selecting appropriate formats and optimizing for size and quality.

Tools like Photopea were used to adjust image sharpness, contrast, and lighting to improve their quality for the website.

Code Structure

The website was developed with a structured HTML and CSS codebase, aiming for clarity and maintainability.

Challenges and Learnings

One of the challenges faced was the initial quality of the product images, which were taken with a 16-megapixel mobile camera, resulting in some blurriness. Efforts were made using Photopea to improve sharpness and adjust other details, but the initial quality limited the final outcome.

Despite this, the project provided valuable experience in structuring a website, applying CSS for design, working with different image formats for web optimization, and understanding the importance of initial image quality. While JavaScript functionality was intended, it was not implemented within the project timeframe.
