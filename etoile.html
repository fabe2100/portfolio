This project simulated a real-world web development scenario, where our group took on two roles: client and web agency. We created a website based on another group's brief and also provided a brief for another group to develop.

My Roles and Responsibilities

  • Acted as part of a team to define a website brief, including company information, target audience, website purpose, and content requirements.
  • Collaborated with my group to develop a functional website based on a provided brief.
  • Contributed to the design and implementation of the website using HTML and CSS.
  • Participated in the client presentation, showcasing the developed website and explaining design choices.
  • Provided individual reflection on the project, analyzing design decisions and the development process.

Project Highlights

Part 1: Brief Creation (Client Role)

Our group developed a brief for a fictional restaurant called "Etoile." The brief included:

  • Company description and target audience.
  • Website purpose: to provide information (opening hours, menu, address) and facilitate online bookings.
  • Required pages: Home, Menu, Book a Table, Contact Us, About Us.
  • Graphic guidelines: logo, color scheme, typography.
  • Content: Text and images (AI-generated and sourced with attribution).

Part 2: Website Development (Web Agency Role)

Our group developed a website based on another group's brief. Key aspects of this phase included:

  • HTML and CSS development to create the website structure and design.
  • Implementation of navigation and page layouts.
  • Focus on user experience and accessibility.
  • Development of a high-fidelity prototype.
  • Client presentation via a recorded video.

Reflection

This project provided valuable insights into the roles of both client and web developer. It emphasized the importance of clear communication through the brief, the practical application of HTML and CSS, and the need for user-centered design. The experience highlighted the collaborative nature of web development and the importance of considering client requirements while creating a functional and engaging website.
